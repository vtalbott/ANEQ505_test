### Mac 

1. Install VS code found here: https://code.visualstudio.com/download
2. Open Obsidian
3. Go to community plugins
4. Find and install Open vault in VS Code
5. Enable Open Vault in VS Code plug in
6. Now open VS code 
7. Open the command panel (CMD + Shift + P)
8. Type  ``` Shell Command: Install 'code' command in PATH ```
9. You will get a confirmation message
10. Open a terminal
11. type ``` which code ```
12. You should get something that looks like ```/usr/local/bin/code``` copy this
13. Go back to obsidian and open the settings for the Open vault in VS code plug in 
14. Under the Open via `code` CL settings paste `/usr/local/bin/code` in front of what was already there so it should look like this:
`/usr/local/bin/code "{{vaultpath}}" "{{vaultpath}}/{{filepath}}"`
15. Right click a note you want to open in VS code ex homework 1 then select open in VS code
16. Once VS code is open go to the bottom right hand corner where it says Markdown and change the language to powershell
17. Make sure to save any changes before exiting out of VS code.


Windows Users
1. Install VS code found here: https://code.visualstudio.com/download
2. Go to community plugins and install Open Vault in VS Code 
3. Enable the plugin 
4. Right click a note you want to open in VS code ex homework 1 then select open in VS code
5. Once VS code is open go to the bottom right hand corner where it says Markdown and change the language to powershell
6. This will show you if there are any hidden spaces in your code
7. You can use mark down format in VS code to add to your notes and once you save the file and exit Obsidian will automatically update the note. 
8. Make sure to save any changes before exiting out of VS code.
